Instalasi

Cara install

  • Pertama-tama, ingatlah untuk mengaktifkan Sumber Tak Dikenal (Unknown Sources) dari setelan Keamanan perangkat Anda, untuk mengizinkan pemasangan Aplikasi bukan pasar (dengan aman mengabaikan dialog berikutnya);
  • Aplikasi sementara yang tidak aktif seperti Halo, Screenfilter, Twilight, manajer sandi LastPass atau yang serupa (aplikasi yang "menarik aplikasi lain"), jika tidak tombol "pasang" tidak akan berfungsi;
  • Untuk menginstal aplikasi, klik pada pesan bar pemberitahuan Anda untuk download selesai (jika menggunakan aplikasi unduhan sistem, jika tidak, ini mungkin berbeda: misalnya jika Anda menginstal manajer file, Anda dapat browse ke folder download Anda dan menginstal APK YouDO secara langsung, Klik pada file).

Pemakaian

Cara Pakai Aplikasi

  • Untuk download video anda tinggal cari video yang di inginkan lewat browser atau aplikasi youtube yang ada di handphone
  • Setelah menemukan video yang di inginkan klik share dan pilih share via Youdo
  • Pilih ukurun video atau format yang diinginkan, dengan klik tombol yang muncul di aplikasi Youdo

Fitur

Fitur Dalam Aplikasi

  • Aplikasi ini hanya di khususkan mendownload video atau audio dari youtube
  • Aplikasi ini menyediakan beberapa pilihan (jika tersedia) ukuran video

Configuration There are 2 configuration options set via extract: extract(youtubeLink, /*parseDashManifest*/ true, /*includeWebm*/ true); parseDashManifest The dash manifest contains dash streams and usually additionally the higher quality audio formats. But the main difference is that dash streams from the dash manifest seem to not get throttled by the YouTube servers. If you don't use the dash streams at all leave it deactivated since it needs to download additional files for extraction. Known issue: the dash manifest can't be parsed for signature enciphered videos includeWebm If set to false it excludes the webm container format streams from the result.
